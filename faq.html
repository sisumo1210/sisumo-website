ご予約・キャンセル

Q どうやって予約できますか？ +
A
このサイトの仮予約フォームに日程・人数・連絡先をご入力ください。空室を確認の上、メールにてご連絡いたします。その後、クレジットカード決済リンクをお送りし、決済完了をもって予約確定となります。
Q キャンセルポリシーを教えてください。 +
A
以下のキャンセル料が発生します。
キャンセル日 キャンセル料
31日前まで無料
30〜15日前宿泊料金の30%
14〜8日前宿泊料金の50%
7〜2日前宿泊料金の80%
前日・当日・無連絡宿泊料金の100%
Q 空き状況はどこで確認できますか？ +
A
ご予約フォームのカレンダーにてご確認いただけます。ご不明な場合はお問い合わせください。
Q 長期滞在はできますか？ +
A
はい、対応しています。1週間以上の長期滞在もお気軽にご相談ください。キッチン・洗濯設備が完備されているため、ワーケーションや移住体験にも適しています。

チェックイン・チェックアウト

Q チェックインの方法を教えてください。 +
A
セルフチェックインに対応しています。ご予約確定後、チェックイン方法（キーボックスの番号等）をメールにてご案内いたします。スタッフは現地に常駐しておりません。
Q チェックイン・チェックアウトの時間は？ +
A
チェックイン：15:00〜 ／ チェックアウト：〜10:00です（SISUMO昆布は〜11:00）。時間外のご対応はご相談ください。
Q 早めに到着した場合、荷物を預けられますか？ +
A
チェックイン前の荷物預かりについてはご相談ください。状況によって対応いたします。

施設・設備

Q BBQはできますか？ +
A
SISUMO鈴川ではBBQプランをご用意しています（食材・機材一式込み）。ご予約時にオプションとしてお申し込みください。SISUMO昆布ではBBQの設備はございません。
Q Wi-Fiはありますか？ +
A
はい、両施設とも高速Wi-Fiを完備しています。リモートワークやワーケーションにもご利用いただけます。
Q 駐車場はありますか？ +
A
はい、両施設とも無料の駐車場があります。台数はお問い合わせください。
Q 喫煙はできますか？ +
A
室内は全面禁煙です。屋外での喫煙はご相談ください。

ペット

Q ペットを連れて行けますか？ +
A
SISUMO鈴川はペット同伴可能です（大型犬も可）。SISUMO昆布についてはお問い合わせください。
Q ペット同伴の追加料金はありますか？ +
A
はい、別途ペット受入料金がかかります。
ペット受入料金（特別清掃費）：4,000円（税込）／ 頭・泊
最大2頭まで同伴可能です。ご予約時にお申し込みください。
Q ペット同伴時の注意事項を教えてください。 +
A
以下の点をご確認の上、ご利用ください。
  • 室内ではマナーパンツ（またはマナーベルト）の着用をお願いしています。
  • ベッド・布団の上には乗せないでください。
  • 長時間のお留守番はご遠慮ください。
  • 排泄物は飼い主様が処理してください。
  • 破損・汚損があった場合は実費をご負担いただきます。
  • ワクチン・狂犬病予防接種済みであること。
  • 攻撃性のある犬は宿泊できません。

お支払い

Q 支払い方法を教えてください。 +
A
クレジットカード（Visa・Mastercard・JCB・American Express等）でのお支払いが可能です。ご予約確定後、決済リンクをメールにてお送りします。
Q 現地での現金払いはできますか？ +
A
現地での現金払いには対応していません。事前のクレジットカード決済のみとなります。

その他

Q 近くにコンビニやスーパーはありますか？ +
A
車で数分以内にコンビニがあります。大型スーパーへは車で15〜20分程度です。食材の買い出しは事前にお済ませいただくことをおすすめします。
Q 領収書の発行はできますか？ +
A
はい、対応しています。ご希望の方はお問い合わせください。
